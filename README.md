# HTTP-flood
HOW TO USE THE SCRIPT
------------------------------------------------------------
FOR WINDOWS:
download zip 
extract on desktop
open a cmd in HTTP-flood 
type : python http-flood.py 


------------------------------------------------------------
FOR LINUX :
open terminal and type this :
git clone https://github.com/Nzxt1q/HTTP-flood/
cd HTTP-flood
python3 http-flood.py

------------------------------------------------------------
FOR TERMUX:
git clone https://github.com/Nzxt1q/HTTP-flood/
bash HTTP-flood 
python3 http-flood.py
